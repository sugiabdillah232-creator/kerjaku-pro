Privacy Mode & Backup Lokal

Data tidak dikirim ke server

Aplikasi ini berjalan sebagai kalkulator pribadi. Data absensi, gaji, konfigurasi, dan riwayat hanya tersimpan di browser/perangkat pengguna. Tidak ada admin yang bisa membaca data karena tidak ada database pusat.

Gunakan backup terenkripsi agar file cadangan tidak mudah dibaca orang lain. Simpan password backup dengan aman karena tidak bisa dipulihkan jika lupa.
